# QuantScope React Frontend

A modern React frontend for QuantScope stock analysis platform.

## Tech Stack

- React 18 + TypeScript + Vite
- Tailwind CSS + Zustand + React Router
- TanStack Query + ECharts

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker-compose up -d
```

Frontend available at http://localhost:3000
